Sistema de Biblioteca em C
Este é um projeto de gerenciamento de biblioteca desenvolvido em linguagem C. O objetivo é controlar o cadastro de livros, usuários e o fluxo de empréstimos através do terminal.

O que o sistema faz?
Cadastro completo: Registra livros (título, autor, ano) e usuários.

Controle de Empréstimos: Gerencia a disponibilidade dos livros e as datas de devolução.

Salvamento Automático: Utiliza arquivos .txt para que os dados não sejam perdidos ao fechar o programa.

Busca e Relatórios: Permite encontrar livros rapidamente e ver estatísticas de uso.

Tecnologias Utilizadas
Linguagem C

Structs (Estruturas de dados para Livros, Usuários e Empréstimos)

Manipulação de Arquivos (Leitura e escrita em arquivos de texto)

Lógica Modular (Organização do código em funções)

Como usar
Compile o arquivo:
gcc "Trabalho Prático - Sistema de Biblioteca com Usuários e Empréstimos.c" -o biblioteca

Execute o programa:
./biblioteca
